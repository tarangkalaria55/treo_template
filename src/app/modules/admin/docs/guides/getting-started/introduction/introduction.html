Introduction

Thank you for choosing Treo.

Treo is a multi-purpose, multi-layout Angular admin template featuring custom made Application and Page layouts, UI elements and UX best practices.

In this documentation, you will find starter guides and API documentation of the custom made Treo components.

Main libraries

Treo powered by 3 main libraries; Angular, Angular Material and TailwindCSS:

  • Angular is the core of the Treo. This version only works with Angular.
  • Angular Material provides the main UI components such as Buttons, Form fields, Tabs, Progress bars and more.
  • TailwindCSS is the core of the style configuration and it provides utility classes for almost every CSS rule available.
